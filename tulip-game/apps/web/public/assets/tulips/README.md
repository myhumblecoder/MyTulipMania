# Tulip Assets

Place your 6 tulip PNG images here:

- `common.png` - Light green tulip (50% drop rate)
- `uncommon.png` - Royal blue tulip (30% drop rate)  
- `rare.png` - Purple tulip (15% drop rate)
- `epic.png` - Gold tulip (4% drop rate)
- `legendary.png` - Hot pink tulip (0.9% drop rate)
- `mythic.png` - Magenta tulip (0.1% drop rate - HOLY SHIT MOMENT!)

## Recommended Image Specs

- **Size**: 100×100px to 200×200px (will be scaled by Phaser)
- **Format**: PNG with transparency
- **Style**: Simple, colorful, recognizable tulip silhouettes
- **Color match**: Match the rarity colors for visual consistency
  - Common: `#90EE90` (light green)
  - Uncommon: `#4169E1` (royal blue)
  - Rare: `#9370DB` (purple)
  - Epic: `#FFD700` (gold)
  - Legendary: `#FF1493` (hot pink)
  - Mythic: `#FF00FF` (magenta)

## Quick Asset Generation

You can:
1. Use AI art generators (DALL-E, Midjourney) with prompt: "simple colorful tulip icon, [COLOR], transparent background, game asset style"
2. Use Canva or Figma to create simple tulip shapes
3. Find free tulip icons on sites like IconFinder, Flaticon, or The Noun Project
4. Commission a pixel artist for retro-style tulips

Until you add real assets, the game will show placeholder textures during development.
